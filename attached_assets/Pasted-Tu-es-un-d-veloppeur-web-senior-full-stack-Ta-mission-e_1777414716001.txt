Tu es un développeur web senior full-stack.

Ta mission est de créer un site web professionnel complet pour une activité de développement web et solutions digitales.

## 🎯 Objectif du site
Créer un site vitrine moderne qui présente et vend des services de création de :
- sites web simples (HTML / CSS / JS)
- sites vitrines professionnels
- e-commerce
- webapps / outils sur mesure
- solutions digitales (automatisation, gestion, logistique, CRM)

Le site doit être orienté conversion (obtenir des demandes de devis).

---

## 🧱 Stack technique
- Frontend : HTML5, CSS3, JavaScript vanilla (ou framework léger si nécessaire)
- Design responsive (mobile-first)
- Code propre et structuré
- Pas de dépendances inutiles
- Animations légères possibles (scroll, hover)
- SEO basique intégré (meta tags, structure H1/H2)

---

## 🎨 Style / design
- Style moderne, minimaliste et professionnel
- Couleurs : fond clair ou sombre sobre + couleur accent (bleu ou violet conseillé)
- Typographie lisible et moderne
- Sections bien espacées
- Look “agence digitale premium”

---
